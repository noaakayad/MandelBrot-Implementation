# Mandelbrot Set Implementation in C

This repository contains a C implementation for generating Mandelbrot fractal images.

The project includes several implementation versions, color utilities and sample Mandelbrot output files.

## Contents

```text
MANDELBROT/
  colors.c
  colors.h
  mandels/
    mandel_v0.c
    ...
    mandel_v9.c
  *.mdb
```

## Features

- Mandelbrot iteration and escape-time computation.
- Multiple implementation versions for progressive development.
- Color handling utilities.
- Sample generated outputs.

## Technologies

- C
- Numerical programming
- Fractal generation

## Notes

The repository is useful as a compact example of low-level numerical programming and image generation in C.
